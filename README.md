# Cofit 登入／註冊互動原型

互動式 HTML 原型，涵蓋 Cofit 平台的完整登入、註冊與忘記密碼流程。

🔗 **線上預覽：** https://stanley-droid.github.io/login/

---

## 畫面列表

### 登入
| 畫面 | 說明 |
|------|------|
| 登入 | Email + 密碼輸入 |
| 登入_密碼錯誤 | 密碼長度不足提示 |
| 登入_登入失敗 | 帳號或密碼錯誤提示 |

### 註冊
| 畫面 | 說明 |
|------|------|
| 註冊 | Email + 密碼 + 確認密碼 |
| 註冊_信箱格式錯誤 | Email 格式驗證 |
| 註冊_密碼強度不足 | 密碼需含英文字母及數字，至少 8 位 |
| 註冊_密碼不一致 | 兩次密碼不符 |
| 註冊_信箱已被使用 | 重複信箱提示 |
| 註冊_驗證信已寄出 | 寄送驗證信確認頁 |
| 註冊_連結過期 | 驗證連結失效提示 |
| 註冊_完成 | 註冊成功頁 |

### 忘記帳號
| 畫面 | 說明 |
|------|------|
| 忘記帳號 | 輸入手機號碼 |
| 忘記帳號_OTP | 輸入驗證碼（輸入 `000000` 觸發錯誤） |
| 忘記帳號_OTP 錯誤 | 驗證碼錯誤提示 |
| 忘記帳號_帳號顯示 | 顯示對應帳號 |

### 忘記密碼
| 畫面 | 說明 |
|------|------|
| 忘記密碼 | 輸入 Email |
| 忘記密碼_連結已發送 | 重設信件寄出確認 |
| 忘記密碼_設定新密碼 | 輸入新密碼 + 確認密碼 |
| 忘記密碼_重設成功 | 密碼重設完成頁 |

### 設定頁（側邊欄）
| 畫面 | 說明 |
|------|------|
| 設定_基本資料 | 個人資料編輯（姓名、信箱、手機等） |
| 設定_履歷資料 | 專業證照、體適能及長照相關認證 |

---

## 操作說明

開啟 `index.html` 或前往[線上預覽](https://stanley-droid.github.io/login/)，畫面右下角有**原型導覽**面板，點擊任一按鈕可直接跳至對應畫面。

### 互動邏輯

- **登入**：密碼少於 8 位 → 密碼錯誤畫面；否則 → 登入失敗畫面
- **註冊**：Email 格式、密碼強度（至少 8 位含英文+數字）、密碼一致性驗證；輸入 `used@test.com` 觸發信箱已使用錯誤
- **忘記帳號 OTP**：輸入 `000000` 觸發驗證碼錯誤
- **設定側邊欄**：點擊基本資料 / 履歷資料可切換頁面
